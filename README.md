# Vue

> Vue.js是一个用于创建用户界面的开源JavaScript框架，也是一个创建单页应用的Web应用框架。2016年一项针对JavaScript框架的调查表明，Vue有着89%的开发者满意度。在GitHub上，该项目平均每天能收获95颗星，为GitHub有史以来星标数第3多的项目。
